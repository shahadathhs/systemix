# @systemix/token

A cryptographically secure token generator for API keys, session tokens, CSRF tokens, and more. Includes encoding utilities and a signed-token module (coming soon).

## Installation

```bash
pnpm add @systemix/token
```

## Basic Usage

```typescript
import { generateToken } from '@systemix/token';

// Default: 32 bytes, hex encoding
const token = generateToken();
// e.g. "a1b2c3d4e5f6..."
```

## Options

| Property     | Type     | Default | Description                                  |
| ------------ | -------- | ------- | -------------------------------------------- |
| `byteLength` | `number` | `32`    | Number of random bytes (1–1024)              |
| `charset`    | `string` | `'hex'` | `hex`, `base64`, `base64url`, `alphanumeric` |
| `count`      | `number` | `1`     | Number of tokens to generate (1–10)          |

## Examples

### API key (hex)

```typescript
const apiKey = generateToken({
  byteLength: 32,
  charset: 'hex',
});
```

### Session token (URL-safe)

```typescript
const sessionToken = generateToken({
  byteLength: 24,
  charset: 'base64url',
});
```

### Multiple tokens

```typescript
const tokens = generateToken({
  byteLength: 16,
  charset: 'alphanumeric',
  count: 5,
});
// Returns string[]
```

## Subpath Imports

Import only what you need:

```typescript
// Token generation + encoding utilities
import {
  generateToken,
  bytesToHex,
  bytesToBase64,
} from '@systemix/token/token';

// Shared crypto (getRandomBytes, getRandomInt)
import { getRandomBytes } from '@systemix/token/shared';

// Signed tokens (encode/decode/verify — coming soon)
import { encodeSigned, verifySigned } from '@systemix/token/signed';
```

## Encoding Utilities

| Function                     | Description                 |
| ---------------------------- | --------------------------- |
| `bytesToHex(bytes)`          | Encode bytes to hex string  |
| `bytesToBase64(bytes)`       | Encode bytes to base64      |
| `bytesToBase64Url(bytes)`    | URL-safe base64, no padding |
| `bytesToAlphanumeric(bytes)` | Encode to A–Z, a–z, 0–9     |
